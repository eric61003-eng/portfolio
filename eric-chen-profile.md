## 專業簡介（正式版）

具備空間設計、品牌活動與視覺創作的跨領域執行能力，服務品牌橫跨精品時尚、汽車、科技、醫藥與娛樂產業。

**空間與展覽設計**：負責 INFINITI、MASERATI、MediaTek、GSK、Oral-B 等品牌展覽空間與體驗裝置的規劃設計，從概念發展至現場執行全程參與。

**品牌活動執行**：參與 GUCCI、LOUIS VUITTON、BVLGARI、DIOR、FERRAGAMO 等精品品牌之新品發表、快閃店及晚宴活動，以及 VOLKSWAGEN、FERRARI、HEINEKEN 等品牌活動的視覺統籌與空間規劃。

**視覺設計與 AI 創作**：主導 STARLUX、Imperial Tobacco、Acer 等品牌 Key Visual 設計，並運用 AI 生成技術（ComfyUI）開發動態視覺內容，應用於實際品牌提案與專案交付。

## 個人作品集網站說明

**個人作品集網站｜Portfolio Website**

自主開發的個人作品集網站，以純 HTML / CSS / JavaScript 從零建構，無框架依賴。採深色極簡設計語言，整合空間設計、品牌活動、AI 生成視覺、平面視覺四大類別，透過動態影片輪播、Lightbox 圖庫瀏覽器與流暢頁面切換呈現專案內容。部署於 Vercel，透過 GitHub 進行版本控制與持續更新。

**Personal Portfolio Website**

Self-built portfolio site developed from scratch using HTML, CSS, and JavaScript — no frameworks. Features a dark minimalist UI with four project categories (Spatial Design, Brand Events, AI Creative, Key Visual), an animated video reel system, a custom lightbox gallery viewer, and smooth transitions. Deployed on Vercel with GitHub version control.
